# 📱 Роҳнамои Instagram Bot - Қадам ба қадам

## 🔑 ЧИ ЛОЗИМ АСТ (Ҳама БЕПУЛ)

| Чиз | Аз куҷо | Вақт |
|-----|---------|------|
| Google Gemini API Key | aistudio.google.com | 2 дақиқа |
| Meta Developer Account | developers.facebook.com | 5 дақиқа |
| Instagram Business Account | instagram.com | Дорӣ |
| Railway (Hosting) | railway.app | 3 дақиқа |

---

## 📋 ҚАДАМ 1: Gemini API Key гиред (БЕПУЛ)

1. Ба **https://aistudio.google.com** равед
2. Google аккаунт бо воридшавед
3. "Get API Key" → "Create API Key" занед
4. Калидро нусха гиред ва ба `.env` ворид кунед

---

## 📋 ҚАДАМ 2: Meta App созед

1. **https://developers.facebook.com** равед
2. "My Apps" → "Create App" занед
3. "Business" интихоб кунед
4. App Name: "PhoneShop Bot" ё дигар ном
5. Instagram-ро ба App пайваст кунед

### Instagram Permissions (ичозатҳо):
- `instagram_basic`
- `instagram_manage_messages`
- `pages_messaging`

---

## 📋 ҚАДАМ 3: Webhook танзим кунед

1. App → Instagram → Webhooks
2. Callback URL: `https://your-app.railway.app/webhook`
3. Verify Token: `my_secret_token_123` (ё худатон иваз кунед)
4. Subscribe: `messages` ✅

---

## 📋 ҚАДАМ 4: Railway-да Deploy кунед

```bash
# 1. Кодро GitHub-га гузоред
git init
git add .
git commit -m "Instagram Bot"
git push origin main

# 2. railway.app → New Project → GitHub Repo интихоб кунед

# 3. Variables (Environment) ворид кунед:
PAGE_ACCESS_TOKEN = xxx
VERIFY_TOKEN = my_secret_token_123
GEMINI_API_KEY = xxx
```

---

## 📋 ҚАДАМ 5: Озмоиш кунед

1. Instagram-и худро кушоед
2. Ба саҳифаи бизнесии худ паём фиристед
3. Бот бояд ҷавоб диҳад ✅

---
